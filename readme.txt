NOME
  netflix_search.sh - realiza uma busca no catálogo netflix

SINÓPSE
  ./netflix_search.sh [OPÇÕES] [TEXTO]

DESCRIÇÃO
  O script *busca_netflix.sh* deve realizar uma busca no catálogo netflix dado no arquivo .csv (de https://www.kaggle.com/shivamb/netflix-shows?select=netflix_titles.csv) de acordo com os critérios dados nas opções. Caso não haja opções será feita uma busca por texto no título, diretor ou cast. Nas opções é possível especificar outros filtros.

EXEMPLO
  ./netflix_seach -n 1 -t rodrigo -l drama

  titulo: Call Me Francis
  categoria: TV Show
  duracao: 1 Season
  International TV Shows, Spanish-Language TV Shows, TV Drama
